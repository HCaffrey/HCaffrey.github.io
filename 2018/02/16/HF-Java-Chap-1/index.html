Java的程序结构

自顶而下看，Java程序是

源文件 -- 类 -- 方法 -- 语句

这样的结构。每一个源文件包含有它自己的类，每个类对象有自己的方法，用来完成相应的任务。而方法的基础就是一行行的语句。
