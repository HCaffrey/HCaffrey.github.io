LeetCode23. Merge k Sorted Lists

Posted on 2018-07-18

Description

Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

Example

My Analysis

Definitely, we need to go through every one of the values on the nodes. But we can make use of the sorted linked list(LLists) by comparing the first unchecked element in every linked lists.

My Solution

This solution uses an ArrayList to store the first unchecked element in every k linked lists. Every time we find the minimum in the ArrayList and add it to the result as a ListNode. Then we add the new ListNode following the last checked ListNode.
The advantage of this algorithm is that it is easy to understand.
The drawback of this solution is that not only is it verbose but it costs time when we are operating on an ArrayList as well.

My Solution version 2.0

This time instead of using ListNode[] and ArrayList<Integer> to navigate, I simply use ArrayList<ListNode> to find the minimum value among the unchecked nodes. In order to take advantage of the Collections.min() method, this program needs to @Override the compare() method.
Actually, this idea comes from Solution 1.

Solution 1

The algorithm use PriorityQueue instead of ArrayList to find out the minimum value among the unchecked nodes.
