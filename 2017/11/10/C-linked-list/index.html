C-linked list

Posted on 2017-11-10

摘要

这篇文章主要写了我用C语音实现链表的过称。

1 问题的描述

完成一段代码，能完成一下的链表功能：

  1. insert
  2. delete
  3. print

2 方法的描述

linked list充分利用了指针的方便。它的结构有点像上一次实现的二叉树结构，是一个自引用结构。
链表的头结点包含三个信息：

  1. key
  2. length
  3. *next

而其余节点有相同结构，但是理论上不包含length这个信息
